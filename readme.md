# epiq

## Local dev

Start dev server in background terminal session:

```bash
$ npm run dev
```

Run app with:

```bash
$ npm run execute
```

## Install (WIP)

```bash
$ npm install --global epiq
```

## CLI

```
$ epiq --help

  Usage
    $ epiq

  Options
    --name  Your name

  Examples
    $ epiq --name=Jane
    Hello, Jane
```
